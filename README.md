# American Sign Language Detection

A _realtime image detection_ app which helps to detect **American Sign Language**.

## About

A **cross communication app** breaking down the *communication barrier between especially able and others*.

### Problem Statement

An app to detect American Signal Language by taking input from the user, either from the phoneâ€™s camera or gallery. After detecting the letter, it should be converted from text to speech.
